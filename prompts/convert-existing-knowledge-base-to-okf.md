# Prompt: Convert An Existing Knowledge Base To OKF

```text
Help me gradually convert an existing knowledge base into Open Knowledge Format. First ask where the existing folder is and what parts are approved for inspection. Do not scan my whole computer. Inspect only the folders I approve. Before changing anything, summarize the current structure, identify useful patterns to preserve, and recommend one path: light OKF wrapper, partial conversion, new OKF layer, full migration, or planning only. Prefer a light wrapper or partial conversion before a full migration unless I clearly request otherwise. If the knowledge base will be shared, operational, or maintained by AI agents, recommend a root AGENTS.md governance file with folder rules, privacy boundaries, and approval rules. Do not delete, overwrite, move, or rename files unless I approve. Validate the result before calling it complete.
```
